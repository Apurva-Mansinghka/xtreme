# xtreme

When we add any new user in django admin panel with correct email id and city, 
that user will receive the email with temperature, description and emoji depicting the weather for the chosen city..

Places to be edited as per your persoanl details : 

settings.py - 
EMAIL_HOST_USER = 'your gmail id'
EMAIL_HOST_PASSWORD = 'password of your gmail id'

To check the working directly runserver and open - 
http://localhost:8000/admin/
Database- Admin username - "xtreme", password  - "adminuser"
